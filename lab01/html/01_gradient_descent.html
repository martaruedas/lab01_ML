Función de Coste y Gradiente¶

El coste de funcin es una relacin entre el nivel de produccin y el coste econmico que implica generarlo.

El gradiente es una funcin de valor vectorial y representa la pendiente de la lnea tangente a la grfica de una funcin.

In [133]:
def compute_cost(theta, x, y): 
    m = len(y) # numero total de observaciones
    y_pred = logistic_function(np.dot(x , theta)) # prediccin de y, que es el producto de X y theta
    error = (y * np.log(y_pred)) + ((1 - y) * np.log(1 - y_pred))
    cost = -1 / m * sum(error)
    gradient = 1 / m * np.dot(x.transpose(), (y_pred - y))
    return cost[0] , gradient

Descenso de Gradiente¶

Uno de los algoritmos de optimizacin ms popular por su uso en el campo de las redes neuronales.

In [135]:
def gradient_descent(x, y, theta, alpha, iterations):
    costs = []
    for i in range(iterations):
        cost, gradient = compute_cost(theta, x, y)
        theta -= (alpha * gradient)
        costs.append(cost)
    return theta, costs
